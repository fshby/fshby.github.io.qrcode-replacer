## 一、技术落地实施计划

### 1.1 项目概述

项目名称：二维码替换到模板工具  
项目类型：纯前端静态应用  
目标用户：社群运营、微商、线下活动组织者  
核心功能：上传模板图和二维码图，自动/手动定位二维码区域，透视变换合成，导出PNG图片，捐赠入口与开发者联系  
项目周期：5周（Phase 1核心功能 + Phase 2体验增强 + Phase 2.1 UI/UX优化 + Phase 2.2 捐赠模块）

## 二、模块化设计文档

### 2.1 模块划分原则

1. 单一职责原则：每个模块只负责一个功能领域
2. 高内聚低耦合：模块内部功能紧密相关，模块间依赖最少
3. 可测试性：模块设计便于单元测试和集成测试
4. 可扩展性：模块接口设计考虑未来功能扩展
5. 技术隔离：算法层与UI层分离，便于技术替换

### 2.3 模块详细设计（已更新）

#### 2.3.1 上传模块（ImageUpload）

功能定义：
- 处理模板图和二维码图的上传
- 支持拖拽上传和点击上传
- 图片格式和尺寸校验
- 图片加载和预览

接口规范：

```typescript
interface ImageUploadResult {
  id: string;
  file: File;
  url: string;
  width: number;
  height: number;
  type: 'template' | 'qr';
}

interface ImageUploadOptions {
  maxSize: number;
  minWidth?: number;
  minHeight?: number;
  allowedTypes: string[];
}

class ImageUpload {
  upload(file: File, type: 'template' | 'qr', options?: ImageUploadOptions): Promise<ImageUploadResult>;
  validate(file: File, options?: ImageUploadOptions): { valid: boolean; error?: string };
  loadImage(url: string): Promise<HTMLImageElement>;
}
```

依赖关系：
- 依赖：无（纯工具模块）
- 被依赖：UploadPanel组件、AppStore

#### 2.3.2 二维码检测模块（QRDetector）

功能定义：
- 检测Canvas中的二维码
- 返回二维码角点坐标
- 验证二维码可读性

接口规范：

```typescript
interface QRCodeCorners {
  topLeft: Point;
  topRight: Point;
  bottomLeft: Point;
  bottomRight: Point;
}

interface Point {
  x: number;
  y: number;
}

interface QRDetectResult {
  success: boolean;
  corners?: QRCodeCorners;
  data?: string;
  error?: string;
}

class QRDetector {
  detect(canvas: HTMLCanvasElement): Promise<QRDetectResult>;
  validate(canvas: HTMLCanvasElement): Promise<boolean>;
}
```

依赖关系：
- 依赖：jsQR库
- 被依赖：CanvasView组件、AppStore

interface TransformMatrix {

依赖关系：

#### 2.7.1 透视变换实现

使用perspective-transform库计算单应性矩阵：

```typescript
import { calculateTransform, applyTransform } from 'perspective-transform';

function computePerspectiveTransform(
  srcPoints: Point[],
  dstPoints: Point[]
): (x: number, y: number) => [number, number] {
  const src = srcPoints.flatMap(p => [p.x, p.y]);
  const dst = dstPoints.flatMap(p => [p.x, p.y]);
  return calculateTransform(src, dst);
}

function computeInverseTransform(
  srcPoints: Point[],
  dstPoints: Point[]
): (x: number, y: number) => [number, number] {
  return computePerspectiveTransform(dstPoints, srcPoints);
}
```

#### 2.7.2 双线性插值实现

```typescript
function bilinearInterpolate(
  imageData: ImageDataWrapper,
  x: number,
  y: number
): RGBA {
  const x0 = Math.floor(x);
  const x1 = Math.min(x0 + 1, imageData.width - 1);
  const y0 = Math.floor(y);
  const y1 = Math.min(y0 + 1, imageData.height - 1);

  const wx = x - x0;
  const wy = y - y0;

  const idx00 = (y0 * imageData.width + x0) * 4;
  const idx10 = (y0 * imageData.width + x1) * 4;
  const idx01 = (y1 * imageData.width + x0) * 4;
  const idx11 = (y1 * imageData.width + x1) * 4;

  return {
    r: Math.round(
      imageData.data[idx00] * (1 - wx) * (1 - wy) +
      imageData.data[idx10] * wx * (1 - wy) +
      imageData.data[idx01] * (1 - wx) * wy +
      imageData.data[idx11] * wx * wy
    ),
    g: Math.round(
      imageData.data[idx00 + 1] * (1 - wx) * (1 - wy) +
      imageData.data[idx10 + 1] * wx * (1 - wy) +
      imageData.data[idx01 + 1] * (1 - wx) * wy +
      imageData.data[idx11 + 1] * wx * wy
    ),
    b: Math.round(
      imageData.data[idx00 + 2] * (1 - wx) * (1 - wy) +
      imageData.data[idx10 + 2] * wx * (1 - wy) +
      imageData.data[idx01 + 2] * (1 - wx) * wy +
      imageData.data[idx11 + 2] * wx * wy
    ),
    a: 255
  };
}
```

#### 2.7.3 图像合成实现

```typescript
function compositeImage(
  templateCanvas: HTMLCanvasElement,
  qrImage: HTMLImageElement,
  dstCorners: Point[]
): void {
  const ctx = templateCanvas.getContext('2d');
  if (!ctx) return;

  const qrData = getImageDataFromImage(qrImage);
  const qrW = qrImage.width;
  const qrH = qrImage.height;

  const srcCorners = [
    { x: 0, y: 0 },
    { x: qrW, y: 0 },
    { x: qrW, y: qrH },
    { x: 0, y: qrH }
  ];

  const invTransform = computeInverseTransform(srcCorners, dstCorners);

  const xs = dstCorners.map(p => p.x);
  const ys = dstCorners.map(p => p.y);
  const minX = Math.floor(Math.min(...xs));
  const maxX = Math.ceil(Math.max(...xs));
  const minY = Math.floor(Math.min(...ys));
  const maxY = Math.ceil(Math.max(...ys));

  const imageData = ctx.getImageData(minX, minY, maxX - minX, maxY - minY);

  for (let y = minY; y < maxY; y++) {
    for (let x = minX; x < maxX; x++) {
      const [sx, sy] = invTransform(x, y);
      if (sx >= 0 && sx < qrW && sy >= 0 && sy < qrH) {
        const color = bilinearInterpolate(qrData, sx, sy);
        const idx = ((y - minY) * (maxX - minX) + (x - minX)) * 4;
        imageData.data[idx] = color.r;
        imageData.data[idx + 1] = color.g;
        imageData.data[idx + 2] = color.b;
        imageData.data[idx + 3] = 255;
      }
    }
  }

  ctx.putImageData(imageData, minX, minY);
}
```

#### 2.7.4 Toast通知实现

```typescript
import { ref } from 'vue';

interface ToastMessage {
  id: number;
  type: 'success' | 'error' | 'warning' | 'info';
  message: string;
  duration: number;
}

const messages = ref<ToastMessage[]>([]);
let messageId = 0;

function showToast(type: ToastMessage['type'], message: string, duration: number = 3000) {
  const id = ++messageId;
  messages.value.push({ id, type, message, duration });
  
  if (duration > 0) {
    setTimeout(() => {
      removeToast(id);
    }, duration);
  }
}

function removeToast(id: number) {
  const index = messages.value.findIndex(m => m.id === id);
  if (index !== -1) {
    messages.value.splice(index, 1);
  }
}

const ToastService = {
  success: (message: string) => showToast('success', message),
  error: (message: string) => showToast('error', message),
  warning: (message: string) => showToast('warning', message),
  info: (message: string) => showToast('info', message)
};
```
